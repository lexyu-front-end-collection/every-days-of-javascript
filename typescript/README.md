# Topics

```bash
deno run xxx.ts
```

`--allow-read`: 允許檔案系統讀取操作
`--allow-write`: 允許檔案系統寫入操作
`--allow-net`: 允許網路請求
`--allow-env`: 允許讀取環境變數
`--allow-run`: 允許運行子程序
`--allow-all` 或 `-A`: 允許所有權限
